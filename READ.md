# Nodejs

 Xcode install 
 - Homebrew를 설치하기 위해선 Xcode가 필요하다.
 Homebrew install 
 - Nodejs를 관리하기 용이하기 위해 Homebrew를 이용한다.
 Nodejs install 
 - $ brew install node
 - $ brew install node@12
 - $ brew unlink node
 - $ brew link node@12
 - $ brew cleanup node
 - $ brew link --force --overwrite node@12
 - $ brew uninstall node
